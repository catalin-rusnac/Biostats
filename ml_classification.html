Gradient Descent

Gradient descent is an optimization algorithm used to minimize the loss function by iteratively moving in the direction of steepest descent.

\[ \theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t) \]

Where:

  • \(\theta_t\): Current parameters
  • \(\alpha\): Learning rate
  • \(\nabla J(\theta_t)\): Gradient of the loss function

Loss Functions

Cross-Entropy Loss

\[ L(y, \hat{y}) = -\sum_{i=1}^n y_i \log(\hat{y}_i) \]

Where:

  • \(y_i\): True label
  • \(\hat{y}_i\): Predicted probability
